# Real-image protocol for BioVisionLab

This document describes how to photograph and organize real dual-culture plate images before running BioVisionLab.

## Goal

BioVisionLab works best when images are collected consistently. Good image collection reduces detection errors and makes measurements more reproducible.

## Recommended image setup

Use the same setup for all plates in one experiment:

- Photograph plates from directly above.
- Keep the camera as parallel to the plate as possible.
- Use even lighting.
- Avoid strong shadows and glare.
- Use a plain dark or neutral background.
- Keep the full Petri dish visible in the image.
- Avoid cropping the edge of the plate.
- Keep image distance and zoom consistent.
- Use the same plate size throughout the experiment.

## Plate orientation

For dual-culture assays:

- The left colony should always be the left inoculation plug.
- The right colony should always be the right inoculation plug.
- Plates should be oriented consistently before photographing.
- If labels are written on the plate, avoid placing them over the fungal colonies.

## Suggested filename format

Use filenames that contain plate number and day.
